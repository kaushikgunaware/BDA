1) Student Record Management using MongoDB
use collegeDB
use collegeDB
Switches to the database named collegeDB.
If the database does not exist, MongoDB creates it automatically when data is inserted.
db.createCollection("students")
db.createCollection("students")
Creates a collection named students.
A collection in MongoDB is similar to a table in SQL.
insertOne()
db.students.insertOne({
    roll_no: "CSE001",
    name: "Gaurav K",
    branch:"CSE",
    year: 2,
    marks: 90
})
Inserts one document into the students collection.
Each document stores student details.
Stored Document
{
   roll_no: "CSE001",
   name: "Gaurav K",
   branch: "CSE",
   year: 2,
   marks: 90
}
insertMany()
db.students.insertMany([
{
    roll_no: "CSE002",
    name: "Atharv K",
    branch:"CSE",
    year: 3,
    marks: 86
},
{
    roll_no: "CSE003",
    name: "Vijay B",
    branch:"CSE",
    year: 2,
    marks: 67
}
])
Inserts multiple documents at once.
find().pretty()
db.students.find().pretty()
find() retrieves all documents.
pretty() displays output in a readable format.
updateOne()
db.students.updateOne(
   {roll_no:"CSE001"},
   {$set:{marks:93}}
)
Updates one document where roll_no = CSE001.
$set modifies the value of marks.
Before Update
marks: 90
After Update
marks: 93
deleteOne()
db.students.deleteOne({roll_no:"CSE003"})
Deletes the document whose roll number is CSE003.
Sorting
Descending Order
db.students.find().sort({marks:-1})
Sorts students by marks in descending order.
-1 means highest to lowest.
Example Output
93
86
Projection
db.students.find(
   {},
   {name:1, marks:1, _id:0}
)
Explanation
{} → selects all documents.
name:1 → display name field.
marks:1 → display marks field.
_id:0 → hide MongoDB generated _id.
Output
{ name: "Gaurav K", marks: 93 }
{ name: "Atharv K", marks: 86 }
Limit
db.students.find().limit(1)
Displays only 1 document from the collection.
Aggregation
db.students.aggregate([
   {
      $group:{
         _id:"$year",
         avg_marks:{$avg:"$marks"}
      }
   }
])
Explanation
aggregate() performs advanced operations.
$group groups documents by year.
$avg calculates average marks.
Example Output
{ _id: 2, avg_marks: 93 }
{ _id: 3, avg_marks: 86 }

4) Product Catalog Management using MongoDB
Use Database
use ProductDB
Create Collection
db.createCollection("Products")
Insert Products
db.Products.insertMany([
{
    productid: 101,
    name: "Laptop",
    category: "Electronics",
    price: 65000,
    stock: 10
},
{
    productid: 102,
    name: "Smartphone",
    category: "Electronics",
    price: 30000,
    stock: 25
},
{
    productid: 103,
    name: "Shoes",
    category: "Fashion",
    price: 2500,
    stock: 40
},
{
    productid: 104,
    name: "Watch",
    category: "Accessories",
    price: 5000,
    stock: 15
},
{
    productid: 105,
    name: "Headphones",
    category: "Electronics",
    price: 3500,
    stock: 20
},
{
    productid: 106,
    name: "T-Shirt",
    category: "Fashion",
    price: 1200,
    stock: 50
}
])
Inserts multiple products into collection.
Display Products
db.Products.find().pretty()
Find Products with Price Greater Than 5000
db.Products.find({price: {$gt: 5000}})
Explanation
$gt means greater than.
Displays products whose price is above 5000.
Example
Laptop
Smartphone
Find Products by Category
db.Products.find({category: "Electronics"})
Displays only electronics products.
Update Product Price
db.Products.updateOne(
    {productid: 105},
    {$set: {price: 4000}}
)
Updates price of Headphones from 3500 to 4000.
Delete Product
db.Products.deleteOne({productid: 106})
Deletes product with ID 106.
Sorting Products
Ascending Order
db.Products.find().sort({price: 1})
1 means ascending order.
Lowest price to highest price.
Descending Order
db.Products.find().sort({price: -1})
-1 means descending order.
Highest price to lowest price.
Projection
db.Products.find(
    {},
    {name: 1, price: 1, _id: 0}
)
Explanation
Displays only name and price.
Hides _id.
Output
{ name: "Laptop", price: 65000 }
Limit
db.Products.find().limit(3)
Shows only first 3 documents.
Sort + Limit
db.Products.find()
           .sort({price: -1})
           .limit(2)
Explanation
Sorts products by highest price.
Displays top 2 expensive products.
Output
Laptop
Smartphone
Aggregation
db.Products.aggregate([
{
   $group:{
      _id: "$category",
      average_price:{$avg:"$price"}
   }
}
])
Explanation
Groups products according to category.
Calculates average price for each category.
Example Output
{ _id: "Electronics", average_price: 33000 }
{ _id: "Fashion", average_price: 1850 }











| Operator       | Meaning                     |
| -------------- | --------------------------- |
| `$set`         | Updates field value         |
| `$gt`          | Greater than                |
| `$avg`         | Calculates average          |
| `$group`       | Groups documents            |
| `sort()`       | Sorts documents             |
| `limit()`      | Restricts number of outputs |
| `find()`       | Retrieves documents         |
| `insertOne()`  | Inserts one document        |
| `insertMany()` | Inserts multiple documents  |
| `updateOne()`  | Updates one document        |
| `deleteOne()`  | Deletes one document        |
